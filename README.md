# Network Troubleshooting Toolkit

A beginner Python-based network troubleshooting utility for macOS.

## Features
- Displays hostname
- Displays local IP address
- Performs ping connectivity tests

## Technologies Used
- Python 3
- macOS Terminal

## How to Run

```bash
python3 network_toolkit.py
